### git基本使用

下载GitHub的东西
```shell
git clone git@github.com:p0ssword/reading-with-iron-and-blood.git
```

添加远程仓库
```shell
git remote add origin git@github.com:p0ssword/reading-with-iron-and-blood.git
```

添加更新的内容描述
```shell
git commit -m "内容描述"
```

push文件上GitHub
```shell
git push origin main:main
```
